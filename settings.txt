## NOTES FOR IMPLEMENTATION:

1. **Database Schema**: Each setting should be stored with:
   - setting_key (unique identifier)
   - setting_value (JSON for complex values)
   - setting_type (string, number, boolean, color, image, url)
   - setting_category (hero, leaders, events, etc.)
   - is_multilingual (boolean)
   - default_value (fallback)
   - description (admin help text)

2. **Admin Panel Categories**:
   - Hero Section
   - Leaders Section  
   - Events Section
   - About Section
   - Photo Gallery
   - Blog Section
   - Video Gallery
   - Global Theme
   - Colors & Typography
   - Layout & Spacing
   - Animations
   - Accessibility
   - Performance
   - Social Media

3. **UI Components Needed**:
   - Color picker for color settings
   - Image uploader for image settings
   - Range sliders for numeric values
   - Toggle switches for boolean settings
   - Multi-language text inputs
   - Preview functionality for visual changes
   - Reset to defaults button
   - Import/Export settings functionality

4. **Validation Rules**:
   - Image files: max size, allowed formats
   - URLs: valid format validation
   - Colors: hex/rgb validation
   - Numbers: min/max ranges
   - Text: character limits

5. **Real-time Preview**: 
   - Settings should update preview in real-time
   - Changes should be saveable as drafts
   - Ability to preview in different themes (light/dark)
   - Mobile/tablet/desktop preview modes

## DATABASE SCHEMA

### 1. Settings Table (Main)
```sql
CREATE TABLE settings (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  setting_key VARCHAR(255) UNIQUE NOT NULL,
  setting_value JSONB,
  setting_type VARCHAR(50) NOT NULL, -- 'string', 'number', 'boolean', 'color', 'image', 'url', 'json'
  category VARCHAR(100) NOT NULL, -- 'hero', 'leaders', 'events', 'about', 'gallery', 'blog', 'videos', 'header', 'footer', 'menu', 'system'
  subcategory VARCHAR(100), -- 'carousel', 'content', 'visual', 'layout', etc.
  is_multilingual BOOLEAN DEFAULT FALSE,
  default_value JSONB,
  description TEXT,
  validation_rules JSONB, -- min, max, required, pattern, etc.
  display_order INTEGER DEFAULT 0,
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  created_by UUID REFERENCES profiles(id),
  updated_by UUID REFERENCES profiles(id)
);

-- Index for performance
CREATE INDEX idx_settings_category ON settings(category);
CREATE INDEX idx_settings_key ON settings(setting_key);
CREATE INDEX idx_settings_active ON settings(is_active);
```

### 2. Setting Translations Table
```sql
CREATE TABLE setting_translations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  setting_id UUID REFERENCES settings(id) ON DELETE CASCADE,
  language_code VARCHAR(5) NOT NULL, -- 'en', 'bn'
  translated_value JSONB NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  UNIQUE(setting_id, language_code)
);

-- Index for performance
CREATE INDEX idx_setting_translations_setting ON setting_translations(setting_id);
CREATE INDEX idx_setting_translations_lang ON setting_translations(language_code);
```

### 3. Menu Items Table
```sql
CREATE TABLE menu_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  parent_id UUID REFERENCES menu_items(id) ON DELETE CASCADE,
  title_en VARCHAR(255) NOT NULL,
  title_bn VARCHAR(255),
  href VARCHAR(500),
  icon VARCHAR(100), -- Icon class or SVG
  display_order INTEGER DEFAULT 0,
  is_active BOOLEAN DEFAULT TRUE,
  is_dropdown BOOLEAN DEFAULT FALSE,
  expand_only BOOLEAN DEFAULT FALSE,
  target VARCHAR(20) DEFAULT '_self', -- '_self', '_blank'
  css_class VARCHAR(255),
  role_visibility VARCHAR(50) DEFAULT 'all', -- 'all', 'authenticated', 'admin', 'guest'
  date_start DATE,
  date_end DATE,
  device_visibility VARCHAR(20) DEFAULT 'all', -- 'all', 'mobile_only', 'desktop_only'
  description_en TEXT,
  description_bn TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  created_by UUID REFERENCES profiles(id),
  updated_by UUID REFERENCES profiles(id)
);

-- Index for performance
CREATE INDEX idx_menu_items_parent ON menu_items(parent_id);
CREATE INDEX idx_menu_items_order ON menu_items(display_order);
CREATE INDEX idx_menu_items_active ON menu_items(is_active);
```

interface LanguageTabsProps {

}

const LanguageTabs: React.FC<LanguageTabsProps> = ({

}) => {

### 2. Multilingual Input Component
```typescript
interface MultilingualInputProps {
  name: string;
  value: { en: string; bn: string };
  onChange: (value: { en: string; bn: string }) => void;
  label: string;
  type?: 'input' | 'textarea';
  placeholder?: { en: string; bn: string };
  required?: boolean;
}

const MultilingualInput: React.FC<MultilingualInputProps> = ({
  name,
  value,
  onChange,
  label,
  type = 'input',
  placeholder,
  required = false
}) => {
  const [currentLang, setCurrentLang] = useState<'en' | 'bn'>('en');
  
  const handleChange = (newValue: string) => {
    onChange({
      ...value,
      [currentLang]: newValue
    });
  };

  return (
    <div>
      <label className="block text-sm font-medium text-gray-700 mb-2">
        {label} {required && <span className="text-red-500">*</span>}
      </label>
      
      <LanguageTabs
        currentLanguage={currentLang}
        onLanguageChange={setCurrentLang}
        hasTranslation={value.en && value.bn}
      />
      
      {type === 'textarea' ? (
        <textarea
          value={value[currentLang] || ''}
          onChange={(e) => handleChange(e.target.value)}
          placeholder={placeholder?.[currentLang]}
          className="w-full px-4 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-transparent"
          rows={4}
        />
      ) : (
        <input
          type="text"
          value={value[currentLang] || ''}
          onChange={(e) => handleChange(e.target.value)}
          placeholder={placeholder?.[currentLang]}
          className="w-full px-4 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-blue-500 focus:border-transparent"
        />
      )}
      
      {/* Translation Status */}
      <div className="mt-2 text-xs text-gray-500">
        <span className={value.en ? 'text-green-600' : 'text-red-500'}>
          English: {value.en ? '✓ Translated' : '✗ Missing'}
        </span>
        <span className="mx-2">|</span>
        <span className={value.bn ? 'text-green-600' : 'text-red-500'}>
          বাংলা: {value.bn ? '✓ অনুবাদিত' : '✗ অনুপস্থিত'}
        </span>
      </div>
    </div>
  );
};
```
